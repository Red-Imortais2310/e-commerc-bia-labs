# Bia Labs – E-commerce de Nutrição de Alta Performance

Plataforma de e-commerce desenvolvida para Bia Labs, especializada em suplementos e produtos de nutrição de alto desempenho.

## Características

- ✨ Design moderno com tema dark/light
- 🛒 Carrinho de compras funcional
- 📱 Interface responsiva
- 🎨 Animações suaves e interativas
- 💰 Cálculo dinâmico de frete grátis

## Tecnologias

- HTML5
- CSS3 (com variáveis CSS)
- JavaScript vanilla

## Como usar

1. Clone este repositório
2. Abra `index.html` em seu navegador
3. Explore os produtos e teste o carrinho

## Estrutura do projeto

```
├── index.html      # Arquivo principal HTML
├── style.css       # Estilos CSS
├── script.js       # Lógica JavaScript
└── README.md       # Este arquivo
```
